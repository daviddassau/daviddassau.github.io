Cut to me, 4 years later, back in school at the Nashville Software School, learning how to make websites where you can DYNAMICALLY display content.

I consider myself extremely lucky to be able to do what I do every day. With all the gifts and pleaseantries that I am afforded in my daily life, I find that there is little room left to complain.

Projects

Capstone Project

This project is a reflection of everything I have learned in the first 6 months at the Nashville Software School. The project demonstrates my front-end capabilities, specifically with JavaScript, jQuery, AngularJS, Firebase, and Bootstrap. My actual capstone project is a movie quote database called You're Gonna Need a Bigger Quote. It allows a user to find specific quotes from any movie, and if they can't find the quote they're looking for, they can submit one.

Contacts Application

For this assignment, we had to use AngularJS to build a simple contacts app, where a user could add, save, and edit their contacts, like they would using their cell phone. A user is required to log in using their Google account, and their contact information is stored using Google's Firebase. The user can also choose to "favorite" a contact, where it will then show up on the favorites page.


Theme Park Map

For this group project, we had to use browserify to build an interactive theme park map for Disney's Magic Kingdom. On page load, a user can see all 7 sections of the park, and a short description of each one. Some of the other features allow the user to read more about certain attractions, see what time a particular show is playing, and use the time picker to see what is happeniing on a specific date.


Movie History

This project was our introduction to AngularJS. We learned about how everything is split up into partials, controllers, and services, and how we should write the code so that they all work together. With the app, you have the ability to search for any movie, just like you would with IMDB. A few of the other main features include being able to save a movie to your own wishlist, as well as rate a movie.

Local Weather

With our local weather application, a user can use the input field to type in their zip code, and find out the weather, as well as other conditions. They also have the ability to see the 3 and 5 day forecast. The application requires a user login, so that the user can save the forecast for a particular day. This project served as our first look into Google's Firebase.
 

Acme Explosives

This project served as great practice for working with JSON data and Promises. We had to come up with different categories of explosives, as well as different types and their details. The application also served as great practice for getting information to dynamically display on the page using a dropdown menu.
